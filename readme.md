## 📝 Описание

**simple_input** — это крошечная библиотека-обёртка над стандартным вводом-выводом Rust. Она позволяет в одно действие получать от пользователя числа и строки, избавляя от необходимости каждый раз писать многословные конструкции.

### Особенности:
- 🎯 **Минимализм** — всего 3 функции для всего
- 🔄 **Готова к использованию** — сразу парсит и возвращает данные
- 📦 **Лёгкая** — ни одной лишней зависимости
- 🔌 **Простая интеграция** — добавьте в `Cargo.toml` и работайте

## 🎮 Как использовать

1. Добавьте библиотеку в проект
2. Вызовите нужную функцию ввода
3. Используйте полученное значение
